A big day for KDE on Linux — **Kubuntu 26.04 LTS "Resolute Raccoon" is officially released**, and we have a fresh batch of KDE application snaps landing in stable to celebrate.

## 🦝 Kubuntu 26.04 LTS is here

Kubuntu 26.04 LTS "Resolute Raccoon" is out and ready for download. Read the full details in the official release notes:

👉 [Kubuntu 26.04 Release Notes](https://www.kubuntu.org/news/kubuntu-26-04-release-notes/)

## New snap stable releases

A collection of KDE applications have landed in stable. Install any of them with:

```bash
sudo snap install <snap_name>
```

Or refresh if you already have them installed:

```bash
sudo snap refresh <snap_name>
```

### KMyMoney 5.2.70

The KDE personal finance manager lands with a significant milestone — this is the **Qt6 port**, making KMyMoney fully modernized for the current KDE stack:

```bash
sudo snap install kmymoney
```

### Cantor 26.04

Cantor, KDE's frontend for mathematical applications, is **back in stable** after an absence. Supports backends including Maxima, R, Python, Julia, and more:

```bash
sudo snap install cantor
```

### Kid3 3.9.7

The KDE audio tag editor for MP3, FLAC, Ogg, and more:

```bash
sudo snap install kid3
```

### KGeoTag 2.0.0

The KDE photo geotagging application:

```bash
sudo snap install kgeotag
```

### Kig 26.04

KDE's interactive geometry application, great for education:

```bash
sudo snap install kig
```

### Kongress

The KDE conference companion app — handy for keeping up with KDE events:

```bash
sudo snap install kongress
```

### Konqueror

The classic KDE web browser and file manager returns to the snap store:

```bash
sudo snap install konqueror
```
